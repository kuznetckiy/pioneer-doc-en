Working with Pioneer Station
=================================

This is how program’s interface looks like when Pioneer is not connected:

.. image:: /_static/images/work_with_pioneer_station.png
	:align: center

**Main menu** is displayed in top part of the window. Here you can update firmware, Pioneer Station and set USB modem connection. The central part has mission start/stop and motors stop buttons. Connection indicator is on the right side. It displays connected drone info. `Code editor`_ and `autopilot setting`_ tabs are described in corresponding sections.

.. _autopilot setting: ../../settings/autopilot_parameters.html

.. _Code editor: pioneer_station_code_editor.html 

.. attention:: Do not run the flight mission with USB cable attached.
